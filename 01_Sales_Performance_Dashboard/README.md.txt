# 📊 Sales Performance Dashboard (Power BI)

## 📝 Overview
An interactive dashboard to analyze monthly sales performance by category, product, and region using KPI cards, filters, and visual insights.

## 📂 Tools Used
- Power BI
- Excel

## 📈 Insights Delivered
- Total revenue and total orders overview
- Top 5 performing products
- Category-wise and month-wise revenue analysis
- Region and category-based filtering

## 🔍 Features
- Dynamic filters (Month, Region, Category)
- KPI cards with Total Revenue, Orders, and Avg Revenue/Order
- Clean visual layout with modern styling

## 📁 Files Included
- `Sales_Data_Project_Ritik.xlsx` – Sample dataset
- `Sales_Dashboard.pbix` – Power BI dashboard file
